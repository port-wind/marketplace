# Cunzhi Plugin Usage

## Core flow

1. Run `Setup Cunzhi`
2. If the wrapper reports `CLI_MISSING`, follow `Install Cunzhi`
3. Run `Check Cunzhi Status`
4. Run `Open Cunzhi Settings`
5. Copy the generated reference prompt and, if needed, the MCP config snippet

## Supported commands

- `node scripts/setup-cunzhi.mjs`
- `node scripts/run-cunzhi.mjs status`
- `node scripts/run-cunzhi.mjs install-guide`
- `node scripts/run-cunzhi.mjs open-settings`
- `node scripts/run-cunzhi.mjs mcp-config`
- `node scripts/run-cunzhi.mjs prompt-workflow`
