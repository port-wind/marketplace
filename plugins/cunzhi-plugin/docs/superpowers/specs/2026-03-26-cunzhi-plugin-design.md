# Cunzhi Plugin Design

## Goal

Create a Cursor-only plugin that helps users work with the upstream Cunzhi project from inside Cursor.

## Scope

- verify `寸止` and `等一下`
- show install guidance
- launch `等一下`
- show MCP config and prompt workflow guidance

## Non-goals

- auto-installing Cunzhi
- replacing the upstream UI
- Claude Code support
