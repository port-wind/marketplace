# Install Cunzhi

1. Do not call plugin-local scripts.
2. Present the recommended macOS Homebrew command exactly:
   - `brew tap imhuso/cunzhi && brew install cunzhi`
3. Also surface the release URL:
   - `https://github.com/imhuso/cunzhi/releases`
4. Remind the user that both `寸止` and `等一下` must be on PATH before Cursor can use the plugin workflows.
