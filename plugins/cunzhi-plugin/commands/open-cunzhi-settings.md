# Open Cunzhi Settings

1. First run `command -v 等一下`.
2. If `等一下` is missing, tell the user to install Cunzhi first.
3. If it exists, run `等一下`.
4. Tell the user that the settings window is where they can review MCP tools and copy the generated reference prompt.
