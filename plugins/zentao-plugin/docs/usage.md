# Usage

## Prerequisites

- `node` and `npm` must be installed locally
- The plugin reuses `~/.config/zentao/config.toml`

## First run

1. Run the `setup-zentao` command
2. If prompted, run the `login-zentao` command
3. Re-run setup to confirm `whoami` succeeds

## Supported flows

- List products
- List bugs for a product
- Query bug details by ID
- List your own active bugs
- Run `zentao self-test`
