# Usage

## Prerequisites

- `node` and `npm` must be installed locally
- The plugin reuses `~/.yapi/config.toml`

## First run

1. Run the `setup-yapi` command
2. If prompted, run the `login-yapi` command
3. Re-run setup to confirm `whoami` succeeds

## Supported flows

- Search interfaces by keyword
- Query interface details by ID
- List interfaces under a category
- Create docs-sync bindings
- Run docs-sync
